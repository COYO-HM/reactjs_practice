## How to run project

1. 해당 프로젝트를 `git clone` 받는다.
2. 터미널을 열어 `npm install` 을 실행
3. `npm start` 로 프로젝트 실행
   [http://localhost:3000](http://localhost:3000)에 보여지게 됩니다.
